Наследуется мало свойств, в основном - свойства шрифта.

Существуют относительные и абсолютные значения свойств:
Font-size у родительского класса 20px
Font-size у класса 50% - это 50% от 20px - 10px

Ссылки

При добавлении ссылок, можно указать атрибут "a href = "..." target со значениями:
_blank Загружает страницу в новое окно браузера.
_self Загружает страницу в текущее окно.
_parent Загружает страницу во фрейм-родитель, если фреймов нет, то это значение работает как _self.
_top Отменяет все фреймы и загружает страницу в полном окне браузера, если фреймов нет, то это значение работает как _self.

Блочные элементы

<p>, <div>, <h2>, <ul>, и др.

Они занимают все доступное место по горизонтали.
Каждый новый блочный элемент начинается с новой строки.
Блочным элементам можно задавать любые размеры: высота, ширина, внутренние и внешние отступы.

Если блок меньше содержимого, есть свойство overflow:(hidden|visible)

Строчные элементы

Строчным элементам нельзя задать ширину или высоту. Их размер зависит от содержимого.
Так же нельзя задвать вертикальные внешние отступы

Строчным элементам можно задавать внутренние отступы, а так же внешние горизонтальные

Если разместить несколько строчных элементов, то они будут с расстоянием между друг другом. Это происходит потому, что браузер распознает элементы "как отдельные слова".

Текст 1 Текст 2 Текст 3

Чтобы этого избежать, нужно задать размер шрифта отдельным строчным элементам, затем все отдельные элементы поместить внутрь одного блокого, размер шрифта которого будет равен 0

Текст 1 Текст 2 Текст 3

HTML CSS
<div class="stroka">
  <span>Текст 1</span>
  <span>Текст 2</span>
  <span>Текст 3</span>
</div> 		span { border: 1px solid black; font-size: 14px; } .stroka { font-size: 0 }

Можно принудительно сделать строчный элемент блочным, добавив в CSS display:block

Можно создавать блочно-строчные элементы.
display:inline-block
Теперь можно задавать размеры и отступы.

Блочно-строчные элементы отличаются от других тем, что они похожи на блочные, но распологаются в одну строку

Float

CSS свойство float указывает, что текущий элемент должен быть изъят из обычного flow (потока) и прижат к левой или правой стороне родительского элемента. Текст и inline элементы будут обтекать такой элемент.

Для того чтобы избежать вываливания из родительского элемента, нужно указать на родительском элементе класс clearfix, с кодом:
.clearfix::after {
clear:both;
display:table;
content: "";
}
